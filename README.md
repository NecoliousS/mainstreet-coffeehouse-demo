# Main Street Coffeehouse Demo

A modern, mobile-responsive website concept for Main Street Coffeehouse & Bakery in Sturgis, MS.

Built as a portfolio piece to demonstrate:
- Mobile-first responsive design
- Modern CSS (Grid, Flexbox, Custom Properties)
- Performance optimization
- User experience focused layout

**Live Demo:** [GitHub Pages Link]

## Features Shown
- Hero section with call-to-action
- Menu display with pricing
- Business information bar
- Future features preview (ordering, booking, loyalty)
- Contact integration ready

## Tech Stack
- HTML5
- CSS3 (no frameworks, pure code)
- Google Fonts
- Responsive design
